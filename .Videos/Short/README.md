# CalcuLab YouTube Short

30-second vertical promo (1080×1920, 9:16) for [calculab.bio](https://calculab.bio).

## Preview

```bash
cd video
npm install
npm run dev
```

Open Remotion Studio and select **CalcuLabShort**.

## Render

```bash
npm run render
```

Output: `out/calculab-short.mp4`

Quick layout check:

```bash
npm run render:still
```

## Scenes

1. **Hook** — Lab math shouldn't mean spreadsheets
2. **Intro** — CalcuLab logo and tagline
3. **Features** — Solution prep, PCR/qPCR, spectrophotometry, cell culture
4. **Demo** — C₁V₁ dilution calculator in action
5. **Benefits** — Free, private, offline PWA
6. **CTA** — calculab.bio

Design follows the CalcuLab "Digital Lab Notebook" system: lab bench gray grid, Precision Cobalt accent, Outfit + Inter typography.
